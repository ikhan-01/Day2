Check if Number is Prime

Problem Statement: Determine whether a given number n is prime or not.

Example Test Cases:
Input: 7 ‚Üí Output: True
Input: 12 ‚Üí Output: False

Brute Force Method

def is_prime_brute(n):
    if n <= 1:
        return False
    for i in range(2, n):
        if n % i == 0:
            return False
    return True
Time: O(n), Space: O(1)

Optimized Method

def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
Time: O(‚àön), Space: O(1)

GCD and LCM

Problem Statement: Given two integers a and b, compute their GCD and LCM.

Example Test Cases:
Input: a = 12, b = 18 ‚Üí GCD = 6, LCM = 36
Input: a = 7, b = 5 ‚Üí GCD = 1, LCM = 35

Brute Force GCD

def gcd_brute(a, b):
    gcd = 1
    for i in range(1, min(a, b) + 1):
        if a % i == 0 and b % i == 0:
            gcd = i
    return gcd
Time: O(min(a, b)), Space: O(1)

Efficient GCD (Euclidean)

def gcd(a, b):
    while b:
        a, b = b, a % b
    return a
Time: O(log min(a, b)), Space: O(1)

LCM using GCD

def lcm(a, b):
    return abs(a * b) // gcd(a, b)
Time: O(log min(a, b)), Space: O(1)

Factorial

Problem Statement: Calculate the factorial of a given number n.

Example Test Cases:
Input: 5 ‚Üí Output: 120
Input: 0 ‚Üí Output: 1

Iterative Method

def factorial_iterative(n):
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result
Time: O(n), Space: O(1)

Recursive Method

def factorial_recursive(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial_recursive(n - 1)
Time: O(n), Space: O(n) for call stack

Sieve of Eratosthenes

Problem Statement: Generate all prime numbers up to a number n using the Sieve of Eratosthenes.

Example Test Cases:
Input: 10 ‚Üí Output: [2, 3, 5, 7]
Input: 20 ‚Üí Output: [2, 3, 5, 7, 11, 13, 17, 19]

Brute Force Prime Generator

def primes_brute(n):
    primes = []
    for num in range(2, n + 1):
        is_prime = True
        for i in range(2, int(num ** 0.5) + 1):
            if num % i == 0:
                is_prime = False
                break
        if is_prime:
            primes.append(num)
    return primes
Time: O(n‚àön), Space: O(n)

Sieve Method

def sieve(n):
    primes = [True] * (n + 1)
    primes[0:2] = [False, False]
    for i in range(2, int(n ** 0.5) + 1):
        if primes[i]:
            for j in range(i * i, n + 1, i):
                primes[j] = False
    return [i for i, is_prime in enumerate(primes) if is_prime]
Time: O(n log log n), Space: O(n)

Fast Power (Exponentiation)

Problem Statement: Calculate x raised to the power n.

Example Test Cases:
Input: x = 2, n = 10 ‚Üí Output: 1024
Input: x = 5, n = 0 ‚Üí Output: 1

Brute Force Method

def power_brute(x, n):
    result = 1
    for _ in range(n):
        result *= x
    return result
Time: O(n), Space: O(1)

Fast Exponentiation

def fast_pow(x, n):
    result = 1
    while n > 0:
        if n % 2 == 1:
            result *= x
        x *= x
        n //= 2
    return result
Time: O(log n), Space: O(1)

Trailing Zeros in Factorial

Problem Statement: Find the number of trailing zeros in n! (factorial of n).

Example Test Cases:
Input: 5 ‚Üí Output: 1
Input: 100 ‚Üí Output: 24

Brute Force (Count actual zeros)

def trailing_zeros_brute(n):
    from math import factorial
    return len(str(factorial(n)).rstrip('0')) - len(str(factorial(n)))
Time: O(n log n), Space: O(n)

Optimized Method

def count_trailing_zeros(n):
    count = 0
    i = 5
    while n // i:
        count += n // i
        i *= 5
    return count
Time: O(log‚ÇÖ n), Space: O(1)

Check Armstrong Number

Problem Statement: Check whether a number is an Armstrong number or not.

Example Test Cases:
Input: 153 ‚Üí Output: True
Input: 123 ‚Üí Output: False

Brute Force (Same as Optimized for this scale)

def is_armstrong(n):
    digits = list(map(int, str(n)))
    power = len(digits)
    return sum(d ** power for d in digits) == n
Time: O(d), Space: O(1) where d is the number of digits
