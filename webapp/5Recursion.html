Factorial

Problem Statement: Calculate the factorial of a number.

Brute-force Recursive Approach

def factorial_recursive(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial_recursive(n - 1)
Time: O(n), Space: O(n)

Optimized Iterative Approach

def factorial_iterative(n):
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result
Time: O(n), Space: O(1)

Fibonacci Series

Problem Statement: Generate Fibonacci numbers.

Brute-force Recursive Approach

def fibonacci_recursive(n):
    if n <= 1:
        return n
    return fibonacci_recursive(n - 1) + fibonacci_recursive(n - 2)
Time: O(2^n), Space: O(n)

Optimized Memoization Approach

def fibonacci_memo(n, memo={}):
    if n in memo:
        return memo[n]
    if n <= 1:
        return n
    memo[n] = fibonacci_memo(n-1, memo) + fibonacci_memo(n-2, memo)
    return memo[n]
Time: O(n), Space: O(n)

Optimized Iterative Approach

def fibonacci_iterative(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
Time: O(n), Space: O(1)

Sum of Digits

Problem Statement: Sum the digits of a number.

Brute-force Recursive Approach

def sum_of_digits(n):
    if n == 0:
        return 0
    return (n % 10) + sum_of_digits(n // 10)
Time: O(log n), Space: O(log n)

Optimized Iterative Approach

def sum_of_digits_iterative(n):
    total = 0
    while n > 0:
        total += n % 10
        n //= 10
    return total
Time: O(log n), Space: O(1)

Print Subsets

Problem Statement: Print all subsets of a string.

Brute-force Recursive Print

def print_subsets(s, curr="", index=0):
    if index == len(s):
        print(curr)
        return
    print_subsets(s, curr, index + 1)
    print_subsets(s, curr + s[index], index + 1)
Time: O(2^n), Space: O(n)

Optimized Return All Subsets

def get_subsets(s, curr="", index=0, result=[]):
    if index == len(s):
        result.append(curr)
        return
    get_subsets(s, curr, index + 1, result)
    get_subsets(s, curr + s[index], index + 1, result)
Time: O(2^n), Space: O(2^n)

Power of 2 Check

Problem Statement: Check if number is power of 2.

Brute-force Recursive Approach

def is_power_of_two(n):
    if n == 1:
        return True
    if n == 0 or n % 2 != 0:
        return False
    return is_power_of_two(n // 2)
Time: O(log n), Space: O(log n)

Optimized Bitwise Approach

def is_power_of_two_bit(n):
    return n > 0 and (n & (n - 1)) == 0
Time: O(1), Space: O(1)

Palindrome Check (Recursive)

Problem Statement: Check if string is palindrome.

Brute-force Using Reversed String

def is_palindrome_reverse(s):
    return s == s[::-1]
Time: O(n), Space: O(n)

Optimized Recursive Two-pointer

def is_palindrome_recursive(s, left=0, right=None):
    if right is None:
        right = len(s) - 1
    if left >= right:
        return True
    if s[left] != s[right]:
        return False
    return is_palindrome_recursive(s, left + 1, right - 1)
Time: O(n), Space: O(n)
