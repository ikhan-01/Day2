Bubble Sort

Problem Statement: Sort an array of integers using the bubble sort algorithm and trace the process step by step.

Example Input: [5, 2, 3, 1, 4]
Expected Output: [1, 2, 3, 4, 5]
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr
Time: O(n^2), Space: O(1)
Initial: [5, 2, 3, 1, 4]
Pass 1: [2, 3, 1, 4, 5]
Pass 2: [2, 1, 3, 4, 5]
Pass 3: [1, 2, 3, 4, 5]

Selection Sort

Problem Statement: Sort an array of integers using selection sort and trace the minimum selections.

Example Input: [5, 2, 3, 1, 4]
Expected Output: [1, 2, 3, 4, 5]
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_index = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        arr[i], arr[min_index] = arr[min_index], arr[i]
    return arr
Time: O(n^2), Space: O(1)
Step 1: Swap 1 with 5 ‚Üí [1, 2, 3, 5, 4]
Step 2: Swap 2 with 2 ‚Üí [1, 2, 3, 5, 4]
Step 3: Swap 3 with 3 ‚Üí [1, 2, 3, 5, 4]
Step 4: Swap 4 with 5 ‚Üí [1, 2, 3, 4, 5]

Insertion Sort

Problem Statement: Sort an array of integers using insertion sort with step-by-step shifting.

Example Input: [5, 2, 3, 1, 4]
Expected Output: [1, 2, 3, 4, 5]
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
Time: O(n^2), Space: O(1)
Insert 2 into [5] ‚Üí [2, 5]
Insert 3 ‚Üí [2, 3, 5]
Insert 1 ‚Üí [1, 2, 3, 5]
Insert 4 ‚Üí [1, 2, 3, 4, 5]

Merge Sort

Problem Statement: Sort an array using the divide and conquer based merge sort algorithm.

Example Input: [5, 2, 3, 1, 4]
Expected Output: [1, 2, 3, 4, 5]
def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr) // 2
        L = arr[:mid]
        R = arr[mid:]
        merge_sort(L)
        merge_sort(R)
        i = j = k = 0
        while i < len(L) and j < len(R):
            if L[i] < R[j]:
                arr[k] = L[i]
                i += 1
            else:
                arr[k] = R[j]
                j += 1
            k += 1
        while i < len(L):
            arr[k] = L[i]
            i += 1
            k += 1
        while j < len(R):
            arr[k] = R[j]
            j += 1
            k += 1
Time: O(n log n), Space: O(n)
Split [5,2,3,1,4] ‚Üí [5,2] and [3,1,4]
Sort and merge ‚Üí [2,5] and [1,3,4] ‚Üí Merge ‚Üí [1,2,3,4,5]

Quick Sort

Problem Statement: Sort an array of integers using the quick sort algorithm by choosing a pivot and partitioning.

Example Input: [5, 2, 3, 1, 4]
Expected Output: [1, 2, 3, 4, 5]
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quick_sort(left) + [pivot] + quick_sort(right)
Time: O(n log n), Space: O(log n) average
Pivot: 5 ‚Üí left: [2,3,1,4] ‚Üí right: []
Next pivot: 2 ‚Üí left: [1] right: [3,4] ‚Üí Final: [1,2,3,4,5]

Bucket Sort

Problem Statement: Sort a list of positive integers using the bucket sort algorithm.

Example Input: [42, 32, 33, 52, 37, 47, 51]
Expected Output: [32, 33, 37, 42, 47, 51, 52]
def bucket_sort(arr):
    if not arr:
        return arr
    bucket_count = 10
    max_value = max(arr)
    buckets = [[] for _ in range(bucket_count)]
    for num in arr:
        index = int(bucket_count * num / (max_value + 1))
        buckets[index].append(num)
    for bucket in buckets:
        bucket.sort()
    return [num for bucket in buckets for num in bucket]
Time: O(n+k), Space: O(n+k)
Input: [42, 32, 33, 52, 37, 47, 51]
Buckets: [[32], [33], [37], [42], [47], [51], [52]]
Merge: [32, 33, 37, 42, 47, 51, 52]

Radix Sort

Problem Statement: Sort a list of positive integers using radix sort based on digit place values.

Example Input: [170, 45, 75, 90, 802, 24, 2, 66]
Expected Output: [2, 24, 45, 66, 75, 90, 170, 802]
def counting_sort(arr, exp):
    output = [0] * len(arr)
    count = [0] * 10
    for i in arr:
        index = (i // exp) % 10
        count[index] += 1
    for i in range(1, 10):
        count[i] += count[i - 1]
    for i in reversed(arr):
        index = (i // exp) % 10
        output[count[index] - 1] = i
        count[index] -= 1
    for i in range(len(arr)):
        arr[i] = output[i]

def radix_sort(arr):
    max_val = max(arr)
    exp = 1
    while max_val // exp > 0:
        counting_sort(arr, exp)
        exp *= 10
Time: O(nk), Space: O(n+k)
Input: [170, 45, 75, 90, 802, 24, 2, 66]
After 1s place: [170, 90, 802, 2, 24, 45, 75, 66]
After 10s place: [802, 2, 24, 45, 66, 170, 75, 90]
After 100s place: [2, 24, 45, 66, 75, 90, 170, 802]

Count Swaps

Problem Statement: Count the number of swaps performed by bubble sort on an array.

def count_swaps(arr):
    swaps = 0
    for i in range(len(arr)):
        for j in range(len(arr) - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swaps += 1
    return swaps
Time: O(n^2), Space: O(1)

Sorting Challenges

Problem Statement: Sort an array based on frequency of elements. If two elements have the same frequency, sort them by value.

# Challenge: Sort based on frequency then value
from collections import Counter

def frequency_sort(arr):
    count = Counter(arr)
    return sorted(arr, key=lambda x: (count[x], x))
Time: O(n log n), Space: O(n)

Mock Interview 1

Problem Statement: Sort an array containing only 0s, 1s, and 2s without using any sorting function.

def sort_colors(arr):
    low, mid, high = 0, 0, len(arr) - 1
    while mid <= high:
        if arr[mid] == 0:
            arr[low], arr[mid] = arr[mid], arr[low]
            low += 1
            mid += 1
        elif arr[mid] == 1:
            mid += 1
        else:
            arr[mid], arr[high] = arr[high], arr[mid]
            high -= 1
    return arr
Time: O(n), Space: O(1)

Mock Interview 2

Problem Statement: Check if an array is already sorted in ascending order.

def is_sorted(arr):
    for i in range(1, len(arr)):
        if arr[i] < arr[i - 1]:
            return False
    return True
Time: O(n), Space: O(1)
